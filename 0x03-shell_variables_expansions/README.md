This project covers essential shell concepts related to variables, expansions, initialization files, and shell scripting techniques.
Below is a summary of the tasks solved in this project

# Requirements

Shell: bash version 5.x or Ubuntu LTS default

All scripts must:

be executable

pass shellcheck

use exactly the required tools

handle environment variables safely



# Summary of Tasks

0. Alias

Create an alias named ls with value rm *.

1. Hello you

Print hello followed by the value of the user’s $USER.

2. PATH

Add /action to the PATH.

3. New path

Count the number of directories in the PATH.

4. Global variables

List environment variables.

5. Local variables

List all local variables, environment variables, and shell functions.

6. Create local variable

Create a local variable BEST=School.

7. Create global variable

Create a new global variable BEST=School.

8. Arithmetic: POWER DIVIDE

Print the result of a variable to the power of another (base 10).

9. Divide and conquer

Print the result of POWER divide DIVIDE.

10. Love exponent breath

Print the result of BREATH to the power LOVE, using environment variables.

11. Binary → Decimal

Convert a number stored in $BINARY from base 2 to base 10.

12. Combinations

Print all 2-letter lowercase combinations except oo, in alphabetical order.

13. Print float

Display a number from $NUM with two decimal places.

100. Decimal → Hexadecimal

Convert a decimal number in $DECIMAL to base 16 (hex).

101. ROT13

Encode/decode ASCII text using the rot13 cipher.

102. Print odd lines

Print every other line from input, starting with the first line.

103. Water and Stir

Add two numbers encoded in custom bases:

WATER → base water

STIR → base stir

Output sum in base bestchol

This involves:

Translating characters to digits

Converting to decimal

Adding

Converting the sum to octal

Mapping octal digits to bestchol
